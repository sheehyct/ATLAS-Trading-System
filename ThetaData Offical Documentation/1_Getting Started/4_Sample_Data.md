Sample Data Bundle
Overview
The sample bundle lets you test the Theta Data API without needing an account or live credentials.
It includes one day of real market data across several endpoints.

Dataset Type	Example File	Description
Trades	trade.csv	Every trade reported by UTP & CTA.
Quotes (Tick)	quote_tick.csv	Every NBBO quote reported by UTP and CTA.
Quotes (1m)	quote_1m.csv	The most recent NBBO quote for each minute of the session.
OHLCV (1m)	ohlc.csv	OHLCV, trade count and VWAP for each minute of the session.
EOD Snapshot	eod.csv	End of day snapshot of OHLCV and NBBO quote.
